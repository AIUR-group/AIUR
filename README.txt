The Artificial Intelligence Using Randomness (AIUR) is an AI for StarCraft: Broodwar, aiming to be unpredictable thanks to some stochastic behaviors. Please visit http://code.google.com/p/aiurproject/ for further information.

This version uses BWAPI 3.7.4 (see http://code.google.com/p/bwapi/) and Boost 1.53.0 (see www.boost.org). AIUR is written in C++ and under GPL v3 licence. This code is an extension of BWSAL (http://code.google.com/p/bwsal/) under MIT licence. Many thanks to them!

For installation details, please read the INSTALL.txt file. Do not forget to copy the directories.txt file into the bwapi-data directory.

Enjoy AIUR!
